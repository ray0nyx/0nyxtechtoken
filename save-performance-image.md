# How to Add Your Performance Dashboard Screenshot

## Steps to Complete:

### 1. Save Your Image
1. Right-click on the Performance dashboard image you want to use
2. Select "Save image as..." or "Save picture as..."
3. Name the file: `performance-dashboard-screenshot.png`
4. Save it to: `/Users/rayhan/Desktop/WagYu/public/images/demos/`

### 2. Verify the File
The file should be located at:
```
/Users/rayhan/Desktop/WagYu/public/images/demos/performance-dashboard-screenshot.png
```

### 3. Test the Landing Page
1. Start your development server: `npm run dev`
2. Navigate to the landing page
3. Scroll down to the "Powerful Features" section
4. Look for the "Track Performance" card - you should see your new Performance dashboard image
